# L-01 数组

**模块ID**：L-01  
**路由**：`/modules/array`  
**难度**：⭐  
**状态**：P0

## 1. 模块概览

目标：展示数组的顺序存储、随机访问、插入删除导致的元素移动。

## 2. 输入与输出

- 输入：初始数组 `number[]`，操作序列（访问/插入/删除）
- 输出：按操作生成的 `AnimationStep[]`
- 约束：长度 0-50，索引必须在合法区间

## 3. 可视化语义

- `visiting`：被访问元素
- `moving`：因插入/删除发生位移的元素
- `matched`：目标索引或目标值命中

## 4. 首批操作范围（V1）

- `get(index)`
- `insert(index, value)`
- `remove(index)`

## 5. 核心交互

- 选择操作类型并输入参数
- 播放操作步骤
- 操作历史回放

## 6. 验收标准（DoD）

- [ ] 3 类操作都能生成步骤
- [ ] 非法索引有明确报错且不生成步骤
- [ ] 每步 `arrayState` 可追溯，最终状态与操作语义一致
- [ ] 回放结束后可一键重置到初始状态
