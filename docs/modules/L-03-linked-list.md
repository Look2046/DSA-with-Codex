# L-03 链表

**模块ID**：L-03  
**路由**：`/modules/linked-list`  
**难度**：⭐⭐  
**状态**：P0

## 1. 模块概览

目标：可视化链表节点与指针变化，突出插入/删除时的链接重定向。

## 2. 输入与输出

- 输入：初始链表（数组形式）与操作序列
- 输出：`AnimationStep[]`（节点状态 + 指针连线状态）
- 约束：长度 0-30；索引和位置必须合法

## 3. 首批形态（V1）

- 单链表（默认）
- 双链表与循环链表保留为后续扩展

## 4. 首批操作范围（V1）

- `find(value)`
- `insertAt(index, value)`
- `deleteAt(index)`

## 5. 可视化语义

- `visiting`：遍历到的节点
- `new-node`：新建但未挂接节点
- `swapping`：链接调整中的关键节点
- `matched`：命中节点

## 6. 验收标准（DoD）

- [ ] 查找、插入、删除都能生成步骤
- [ ] 每一步连线关系合法（无断链）
- [ ] 删除头/尾/中间节点都可正确回放
- [ ] 非法位置输入被拦截并提示
